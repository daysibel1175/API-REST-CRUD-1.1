# Imágenes de Trilhas

Coloca las imágenes de las trilhas en esta carpeta.

## Ejemplos de rutas para usar en el formulario:

- `/trilhas/trilha1.jpg`
- `/trilhas/trilha2.jpg`
- `/trilhas/trilha3.jpg`

## Nombres sugeridos basados en las imágenes proporcionadas:

1. **trilha1.jpg** - Cañón con río (imagen con río y montañas verdes)
2. **trilha2.jpg** - Cascada (imagen con cascada y vegetación)
3. **trilha3.jpg** - Río con rocas (imagen de río con piedras)

Puedes renombrar las imágenes que subas con estos nombres o usar los nombres que prefieras.
